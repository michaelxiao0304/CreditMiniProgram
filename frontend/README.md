# 贷款小程序前端

## 需要准备的图片资源

在 `images` 目录下需要添加以下图片：

### TabBar 图标 (需要2组)
- `home.png` - 首页未选中状态 (81x81px)
- `home-active.png` - 首页选中状态 (81x81px)
- `user.png` - 我的未选中状态 (81x81px)
- `user-active.png` - 我的选中状态 (81x81px)

### 功能图标
- `heart.png` - 收藏图标 (44x44px)
- `heart-active.png` - 已收藏图标 (44x44px)
- `history.png` - 历史记录图标 (44x44px)
- `feedback.png` - 反馈图标 (44x44px)
- `about.png` - 关于我们图标 (44x44px)

### 占位图
- `default-avatar.png` - 默认头像 (120x120px)
- `bank-default.png` - 默认银行Logo (80x80px)

## 注意事项
- 所有图标建议使用 PNG 格式
- TabBar 图标需要同时准备普通版和高清版 (@2x, @3x)
- 建议使用微信绿色 (#1AAD19) 作为图标主色调
